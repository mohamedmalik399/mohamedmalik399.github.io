Lab #1

Basic Router Configuration & Hostame Set

Explanation: This lab is demonstates how to enter global configuaration mode, change the router name, and secure privileged EXEC mode using and encrypted password.


Router> enable                    
Router# configure terminal
Router(config)# hostname R1
R1(config)# enable secret CCNA
R1(config)# exit
R1# copy running-config startup-config
Lab #2

Configuration an Interface IP Address

Explanation: Assigning an IPv4 address to a GigabitEthernet interface and enabling the port using the no shutdown command.


Router> enable                    
R1# configure terminal
R1(config)# interface GigabitEthernet0/0/0
R1(config-if)# ip address 192.168.1.1 255.255.255.0
R1(config-if)# no shutdown
R1(config-if)# exit
